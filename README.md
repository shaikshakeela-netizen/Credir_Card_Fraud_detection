# 📌 Project Overview

The **Credit Card Fraud Detection System** is an interactive Machine Learning web application developed to identify fraudulent credit card transactions using classification algorithms and imbalance handling techniques.

This project combines:

- 📊 Data Analysis
- 🤖 Machine Learning
- 📈 Data Visualization
- 🌐 Streamlit Web Application

The application allows users to:

✅ Upload transaction datasets  
✅ Perform Exploratory Data Analysis (EDA)  
✅ Train Machine Learning models  
✅ Visualize fraud patterns  
✅ Predict fraudulent transactions in real time  

---

# 🎯 Problem Statement

Credit card fraud has become one of the biggest financial threats in digital transactions. Detecting fraudulent activities is challenging because:

- Fraud cases are extremely rare compared to normal transactions
- Datasets are highly imbalanced
- Real-time prediction is required
- Fraud patterns continuously evolve

The objective of this project is to build an intelligent fraud detection system capable of accurately identifying suspicious transactions using Machine Learning techniques.
